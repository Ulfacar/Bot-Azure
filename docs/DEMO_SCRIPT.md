# Демо-скрипт для встречи с отелем

## Что это?

AI-ассистент для отеля в WhatsApp и Telegram. Отвечает гостям 24/7, собирает заявки на бронь, знает все цены и правила. Менеджер видит все диалоги в удобной панели.

---

## Что бот умеет (показать на живом примере)

### 1. Мгновенные ответы на вопросы гостей
- Прайс по сезонам (точная цена по датам)
- Описание номеров, удобства
- Как добраться, трансфер
- Правила заезда/выезда, оплата, отмена

**Демо:** написать боту "Сколько стоит номер на двоих на 15-17 июля?" — бот за 3 секунды даст точную цену по сезону.

### 2. Сбор заявок на бронирование
- Бот собирает: даты, кол-во гостей, ФИО, телефон
- Формирует заявку и передаёт менеджеру
- Менеджер получает уведомление в Telegram с кнопками

**Демо:** пройти мини-бронирование — бот спросит все данные и передаст менеджеру.

### 3. Админ-панель для менеджера
- Все диалоги в одном месте (WhatsApp + Telegram)
- Перехват чата от бота в любой момент
- Фильтры по статусу (новые, нужна помощь, закрытые)
- Статистика: сколько диалогов, сколько бот решил сам

**Демо:** открыть панель, показать реальные диалоги Тон Азура.

### 4. Самообучение
- Менеджер видит вопрос, на который бот не смог ответить
- Нажимает "Обучить" — бот запоминает ответ навсегда
- База знаний растёт с каждым днём

### 5. Заметки о клиентах
- Гость звонил или приходил лично — менеджер записывает заметку
- Когда гость пишет боту — бот знает контекст

### 6. 3 языка
- Русский, кыргызский, английский
- Бот автоматически отвечает на языке гостя

**"А если много гостей напишут одновременно?"**
